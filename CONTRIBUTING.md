# Contributing

Thanks for your interest in Pi Insights.

## Issues

File an issue for bugs, feature requests, or questions. Screenshots of your report (redacted if needed) are helpful.

## Pull Requests

PRs are welcome for exceptionally powerful optimisations or improvements. Think: meaningfully better insights, significant performance gains, or fixing something genuinely broken.

For simple customisations (different chart styles, tweaking prompts, adding a section for your workflow), fork the repo and modify to your heart's content. That's what the license is for.

## Running locally

```bash
git clone https://github.com/BlazeUp-AI/pi-insights.git
pi install ./pi-insights
/pi-insights
```
